Jingle is a specialized program for creating short, 4-bar, 8-bit "jingles."
It features an instrument rack up to 4 instruments, and it includes a 3-function additive synthesizer. 



*************HOW TO RUN*************

1. Install all the necessary libraries:
	a. pygame
	b. pydub

2. Make sure that all the required Fonts, Images, and Samples are in the runner.py directory
	a. These should be included in the .zip

3. Execute the module manager update command to acquire libraries more easily.



***********************************

4. There are no shortcut commands
